🧪 裝備合成挑戰 (Gear Alchemy Challenge)
這是一個專為國小四年級學生設計的互動式英文學習遊戲。透過與 AI 合作的感覺，讓學生在課堂上透過「合成」魔法裝備的概念，練習基礎英文單字與聽力。

🌟 遊戲特點
零輸入需求：學生僅需點選畫面選項，無需打字或繪圖，適合課堂快速互動。

雙語對照：預設隱藏中文翻譯，鼓勵學生先閱讀英文，老師可視情況點擊顯示。

語音朗讀 (TTS)：點擊喇叭圖示可自動朗讀英文句子，強化學生的聽力與口說模仿能力。

視覺強調：針對教室投影設計，採用高對比色調與超大字體，確保後排學生清晰可見。

🎯 教學目標
詞彙學習：涵蓋自然元素（Fire, Ice, Wind...）與生活用品（Bike, Hat, Shoes...）共 16 個基礎單字。

句型運用：學習形容詞修飾名詞的結構，以及簡單的超能力描述句型。

聽力訓練：利用 Web Speech API 提供的標準美式發音進行聽力練習。
