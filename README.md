🧪 裝備合成挑戰 (Gear Alchemy Challenge)
這是一個專為 國小四年級學生 設計的互動式英文學習遊戲。透過「化學合成」與「冒險裝備」的概念，讓學生在課堂上透過簡單的點擊互動，感受與 AI 合作創造傳說裝備的樂趣。

## 🌟 遊戲核心特色
免輸入、免繪圖：完全符合教學現場需求，學生無需打字或上傳檔案，只需做出「選擇」。

視覺化學習：針對 10 歲孩童 設計，使用大字體、鮮豔色彩與 Emoji 圖標 輔助記憶。

即時語音回饋：內建 TTS (Text-to-Speech) 技術，合成成功後自動朗讀英文句子，強化聽力。

雙語階梯引導：預設隱藏中文，鼓勵學生先聽、先讀英文，再點擊按鈕確認理解。

## 🎯 教學目標 (Learning Objectives)
單字擴充：熟練 16 個核心基礎單字（包含自然元素與生活用品）。

聽力訓練：辨識 AI 朗讀出的關鍵字（如：Hot, Cold, Fast, Fly）。

句型運用：潛移默化學習 「形容詞 + 名詞」 的組合方式。

## 🛠️ 課堂執行步驟
### 1. 遊戲準備
老師開啟網頁並投影至大螢幕。

點擊 「START 開始挑戰」 進入實驗室。

### 2. 學生互動
邀請兩位學生上台（或全班投票），分別從 素材 A (Nature) 與 素材 B (Items) 中各選一個單字。

點擊 「MERGE 合成」 啟動魔法動畫。

### 3. 成果驗收
聽力練習：點擊 🔊 喇叭按鈕，請學生模仿 AI 的發音讀出超能力描述。

理解確認：點擊 「Show Translation」 揭曉裝備名稱與能力的中文含意。

## 💻 技術說明與安裝
技術架構：純前端開發（HTML5, CSS3, JavaScript）。

語音技術：使用瀏覽器內建 Web Speech API，不需額外安裝插件。

安裝方式：

下載本專案的 index.html。

直接使用 Chrome 或 Edge 瀏覽器開啟即可運行。
